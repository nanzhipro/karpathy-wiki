# micrograd

**Type:** Open-source project (Karpathy, 2020) — a tiny scalar-valued automatic-differentiation engine in ~100 lines of Python. The entry point of Karpathy's [[zero-to-hero|Zero to Hero]] YouTube series.

## Relevance to this wiki

- Bottom of the [[ramps-to-knowledge]] ladder: `micrograd → nanoGPT → nanochat → llm.c`. Each rung strips one more abstraction or adds one more capability.
- Karpathy's prototype proof that **one page of code can teach backprop end-to-end**. Repeatedly referenced as a template for what [[eureka|Eureka]] wants to build at curriculum scale.
