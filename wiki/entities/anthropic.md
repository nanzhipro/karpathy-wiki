# Anthropic

AI safety company, maker of **Claude** and **[[claude-code|Claude Code]]**. Karpathy's most-cited 2025 reference for agent-grade coding tools.

## Products that appear in the wiki

- **Claude** (Sonnet / Opus / Haiku) — general-purpose model family.
- **[[claude-code|Claude Code]]** — CLI agent for codebases; Karpathy's cited default for agentic coding in 2025.
- **[[model-context-protocol|MCP]]** — open protocol Anthropic introduced for tool/context integration, widely adopted across agents.

## Why it matters in Karpathy's 2025 arc

- **Claude Code is the reference implementation** for the "[[karpathy-x-2025-ai-assisted-coding|autonomy slider]]" Karpathy advocates.
- **MCP wins the integration war.** By late 2025 MCP had become the de facto standard for agent tool protocols.
- **Safety framing aligns with [[verification-gap]] worries.** Karpathy's generation-vs-discrimination concern and Anthropic's "constitutional" bent land in overlapping territory, though their framings differ.
